# Personal Configuration Files

This repository contains my personal configuration files (dotfiles) for various development tools.

## Contents

- **Neovim (`nvim/`)**: My modern Neovim configuration, entirely written in Lua and managed using `lazy.nvim`. It includes a variety of plugins for LSP, formatting, file exploration, and more.
- **Starship (`starship/`)**: My configuration for the [Starship](https://starship.rs/) cross-shell prompt.
